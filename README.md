# Simulateur SHDL

Simulateur de circuits logiques en SHDL. Projet long N7 1SN, équipe TOB.

## Lancer

Télécharger le `.jar` de son OS dans [`livrables/jars/`](livrables/jars/), puis `java -jar shdl-simulateur-linux.jar` (ou `-windows`).

Pré-requis : JDK 25+ (JEP 492 `super(...)` après instructions locales).

## Arborescence

`src/` et `editeur/` contiennent le code applicatif JavaFX, `assets/` les icônes, `livrables/` les rapports et jars du rendu, `ORGANISATION/` les comptes-rendus. Le build Gradle vit sur la branche `GUI`.
